关于 JSON 格式化

什么是JSON？

JSON（JavaScript Object Notation）是一种轻量级的数据交换格式。它基于JavaScript的一个子集，采用完全独立于编程语言的文本格式来存储和表示数据。

JSON易于人阅读和编写，同时也易于机器解析和生成，是现代Web应用中最常用的数据交换格式之一。

使用说明

  • 格式化：美化JSON数据，添加缩进和换行
  • 验证：检查JSON语法是否正确
  • 压缩：移除所有空格和换行，压缩JSON
  • 语法高亮：不同JSON元素使用不同颜色显示
  • 折叠功能：点击对象/数组前的按钮可折叠内容
  • 错误定位：JSON格式错误时会显示具体位置
